Relay

Relay is a self-hostable, hackable LLM web environment that can be extended with your own tools and skills. It is for programmers, AI engineers, hackers, and anyone who wants their own AI environment with the option to extend it with code.

Built on Roda, Sequel, Falcon, llm.rb, HTMX and web sockets.

Features

Extensible tools

Add your own Ruby tools to ~/.config/relay/tools/ and they are automatically available to the LLM.

MCP support

Connect MCP servers for GitHub, Forgejo, filesystem access, or any MCP-compatible service. Forgejo support expects and recommends forgejo-mcp from codeberg.org/goern/forgejo-mcp.

Multiple providers

OpenAI, Anthropic, Google Gemini, DeepSeek, xAI, AWS Bedrock, and any OpenAI-compatible endpoint.

Streaming responses

Real-time streaming via web sockets — output appears as it is generated.

Powered by llm.rb

Built on llm.rb — the same runtime for tools, agents, MCP, A2A, skills, and streaming.

Self-hosted

Install as a RubyGem and run on your own infrastructure. No third-party dependencies beyond your provider API key.

Getting started

Install — gem install relay.app
Configure — relay setup
Serve — relay start
